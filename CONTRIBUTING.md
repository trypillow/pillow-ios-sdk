# Contributing

Thanks for your interest in `pillow-ios-sdk`.

This repository is a public release mirror. The monorepo is the source of truth for implementation work, so direct pull requests are not part of the normal contribution flow.

What is welcome here:

- Bug reports
- Integration questions
- Feature requests

If you need help, open a GitHub issue in this repository. For account-specific or product support, use the contact details in [SUPPORT.md](SUPPORT.md).
